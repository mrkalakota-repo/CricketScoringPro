Welcome to the presentation for Gully Cricket Scorer. We are transforming amateur cricket by bringing professional-level data tracking to the street level. Our app ensures that every run, wicket, and match is recorded with precision, creating a digital legacy for players.

Problem Statement

The Chaos of Local Cricket

  • Fragmented Data: Results are lost on paper or forgotten
  • Connectivity Barriers: Apps fail without high-speed internet
  • Community Isolation: No way to find nearby teams or live scores
  • Setup Friction: Existing tools are too complex for casual games
Amateur cricket faces a data gap. Most apps require a stable connection, which is rare on local grounds. There's no hub for discovery, and setup is usually too slow for a spontaneous game.

Solution Overview

A Unified Digital Hub

  • Offline-First: Score anywhere via local SQLite persistence
  • Live Cloud Sync: Real-time ball-by-ball updates via Supabase
  • Proximity Discovery: Find teams within a 50-mile radius
  • Immutable Engine: Robust logic ensures 100% data integrity
We solve this with an offline-first design. Local storage means the game never stops. Cloud sync and GPS proximity create a connected community where finding a game is easy.
